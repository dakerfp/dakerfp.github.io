Python cookbook: argmin & argmax

Tue, Dec 1, 2015 python, argmin, argmax 

def argmin(iter, function):
    return min((f(x), x) for x in iter)[1]

def argmax(iter, function):
    return max((f(x), x) for x in iter)[1]

>>> argmin(range(-100, 100), lambda x: x * x)
0
>>> argmax(range(-100, 100), lambda x: - x * x)
0
>>> argmax([[1, 2, 4], [1], [2, 5], []], len)
[1, 2, 4]

PIL -> Pillow

Thu, Jan 16, 2014 pillow, pil, python, setuptools, virtualenv

Pillow is a PIL fork created to add new features. setuptools support was also added. A more frequent release cycle was also promised. With Pillow you can have PIL as a package dependency in setuptools and virtualenv. That means less clutter and robustness for us. Pillow allows you to continue to use import PIL, so there is no need to change your current PIL related code. 0 migration overhead. Archlinux already dropped support for PIL in favor of Pillow.
